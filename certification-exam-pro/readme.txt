A comprehensive certification exam practice platform with freemium features.

== Description ==

Certification Exam Pro allows you to create and manage certification practice exams with various question types including multiple choice, multiple select, yes/no, and drag-and-drop questions.

Features:
* Multiple question types
* Media support for questions and answers
* Freemium model with limited free questions
* Progress tracking
* Detailed explanations
* Interactive interface

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/certification-exam-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Certification Exams screen to configure the plugin
